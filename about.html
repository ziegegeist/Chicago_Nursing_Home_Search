What's this site about?

Not all nursing homes are created equal.

This project began with a story in the New York Times by Katie Thomas called “Medicare Star Ratings Allow Nursing Homes to Game the System.” Here’s a video from that story.


Since then Medicare made some improvements to their rating system, but it's still not perfect. The version of this site that you’re on now was built as a proof of concept, but the team that built it is working on a more robust version. This version takes data from data.medicare.gov and puts it on a map, which mimics what’s already available from Medicare’s Nursing Home Compare page. Then, for all Chicago nursing homes there's a link to the Illinois Healthcare and Family Services 2014 Long Term Care Cost Reports. These reports are rich in data.

What we want to do in the future is combine the Medicare data with key performance indicators from the more detailed Cost Reports using an algorithm similar to the one used by Cal Quality Care, but suited for Chicago. Also on our to-do list is to add information about the neighborhoods surrounding the facilities and to make it convenient to see how far away each one is from you on public transportation.

This project was created by volunteers at Chi Hack Night, and if you would like to contribute the best way is come join us one Tuesday evening. You can also find this project on Github.
